## 기본세팅 방법

1. node.js 를 다운받는다.
2. node.js command prompt 에서 ```npm install -g gatsby-cli``` 로 gatsby 를 설치한다.
3. [Gatsby Starters](https://www.gatsbyjs.com/docs/starters/)

4. ```gatsby develop``` or ```gatsby build```
+ 포트번호를 바꾸는건 ```-p 4000``` 옵션으로 가능하다.
